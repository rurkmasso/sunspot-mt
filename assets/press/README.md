# Press logo assets

Drop SVG (preferred) or PNG files here using these exact filenames:

  times-of-malta.svg
  lovin-malta.svg
  malta-today.svg
  tvm.svg
  malta-ceos.svg
  the-shift.svg

The footer marquee (`eeat-footer.js`) auto-detects them. Until a file is
present, the typographic placeholder shows.

Recommended:
  - SVG with a transparent background
  - Single colour version (the CSS strips colour anyway with grayscale)
  - ~120px wide max, scales to 24px height in the strip
